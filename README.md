New version written from scratch
```bash
                 1  2  3  4  5  6  7  1

Major / Ionian  [2, 2, 1, 2, 2, 2, 1, 2]
Dorian          [2, 1, 2, 2, 2, 1, 2, 2]

Lydian          [2, 2, 2, 1, 2, 2, 2, 1]

Minor / Aeolian [2, 1, 2, 2, 2, 1, 2, 2]
Locrian         [1, 2, 2, 2, 1, 2, 2, 2]
```
Each note lenght in semitones
```bash
C D E F G A H
2 2 1 2 2 2 1
```
Example 1
```bash

```
